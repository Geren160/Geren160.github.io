Proyectos

  • Generador de oficios: Analicé el flujo de trabajo existente para identificar oportunidades de optimización y desarrollé una aplicación en Python con Tkinter y CustomTkinter para automatizar la generación de oficios a partir de datos de Google Forms/Sheets, generando documentos en Google Drive mediante plantillas dinámicas. Incluye formateo y manipulación de datos así como una interfaz amigable para usuarios no técnicos a quienes también capacité en su uso. El proyecto optimiza significativamente la eficiencia operativa y reduce errores manuales.

    Tecnologias: Python, Pandas, Openpyxl, python-docx, Tkinter/CustomTkinter, Google Drive API

  • Punto de venta: Desarrollé un sistema integral de punto de venta con gestión de inventario, control de recetas e ingredientes, menú por categorías, carrito de compras y procesamiento de pagos, optimizando la eficiencia operativa y minimizando errores manuales. La aplicación cuenta con interfaz visual interactiva para usuarios, manejo de bases de datos SQLite y herramientas administrativas para productos e ingredientes. El proyecto contempla expansión futura para el análisis de datos de ventas y comportamiento de clientes, orientado a la toma de decisiones estratégicas.

    Tecnologias: Python, Flet, SQLite, Pandas
